# Advanced E-Commerce Microservices Platform

An enterprise-grade e-commerce backend built using Java and Spring Boot.

## Project Goal

The objective of this project is to learn Spring Boot from scratch by building a real-world e-commerce application. The project will gradually evolve from a monolithic application into a scalable microservices architecture using Spring Cloud and other enterprise technologies.

## Planned Features

- Product Management
- User Authentication
- Shopping Cart
- Order Management
- Inventory Management
- Notification Service
- RESTful APIs

## Planned Technologies

- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Spring Security
- JWT
- Redis
- RabbitMQ
- Spring Cloud Gateway
- Eureka Service Discovery
- Resilience4j
- Docker
- Prometheus
- Grafana
